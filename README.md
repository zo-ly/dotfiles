# My settings

My application settings in sync (OS X) by [Mackup](https://github.com/lra/mackup)

## Usage

```bash
brew install mackup

cd ~

git clone git@github.com:zo-ly/mackup_cfg.git

mackup restore
```

## Back up

```bash
mackup backup

cd ~/mackup_cfg

git # git push to the origin repo
```

## Backed up applications

- Zsh
- Vim
- Git
